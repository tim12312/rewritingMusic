Now about this "project", this is a simple approach to use rewriting rules to stepwise create "music".

The whole thing is based on unique nodes (letters), each node needs to be assigned with a rule that defines how the node will be replaced in the next step, and each node is assigned to a frequents.(The most common frequencies can be looked up at: here)

With the step button you can run the rewriting model to create "music" and with the ▶ button you can listen to your master pice. Moreover you can select different Oscilators(only the once js includes), you can set the speed (not during a audio output), change the volume and loop the output. And if you really want to you can add multiply setups to create different "tracks" and paly them to gether.
